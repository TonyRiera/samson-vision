El problema

Los agentes limitados por APIs de visión pierden errores estructurales. Cambiar de modelo borra el contexto. Los modelos multimodales cuestan más mientras sacrifican profundidad en código y razonamiento.

La respuesta — SAMSON_VISION_PACK

13 campos de texto estructurado que reemplazan la necesidad de enviar imágenes a LLMs costosos. Mismo modelo. Mismas habilidades. Comprensión visual completa a través de texto.

python3 src/samson_vision.py imagen.png --md > pack.md

Flujo con subagentes

El agente principal (DeepSeek Flash v4, sin visión) recibe la tarea con imagen. Antes de delegar, ejecuta Samson Vision CLI (samson-vision imagen.png --md) — pipeline 0% IA. Lee el SVP, luego delega al subagente con visión incorporada (vision_scout) con imagen + SVP embebido. El resultado vuelve al orquestador.

DeepSeek Flash v4 orchestrator → Samson CLI → SVP → vision subagent → back. CLI run by main agent before delegation.

Casos de uso / Use cases

Patrón: orquestador DeepSeek Flash v4 sin visión + Samson CLI + subagente con visión nativa.

Pattern: visionless DeepSeek orchestrator + Samson CLI + vision-capable subagent.

  • Orquestador barato revisa screenshot de UI · Cheap orchestrator reviews UI screenshot
    Principal genera SVP → delega a subagente con visión → fix CSS sin multimodal caro en el orquestador.
  • CI/CD sin modelo de visión en orquestador · CI/CD without vision on orchestrator
    Pipeline corre Samson CLI; orquestador texto-only diffs SVPs; subagente con visión valida regresiones.
  • Subagente Cursor con visión · Cursor vision subagent
    Orquestador embebe SVP + pasa imagen; subagente multimodal opera con ambos.
  • Auditoría de accesibilidad · Accessibility audit
    SVP (OCR + layout + colores) orienta al orquestador; subagente con visión confirma contraste y targets.
  • Portfolio y evidencia · Portfolio / evidence
    Captura prod → SVP → case study versionable; subagente con visión para detalle fino.
  • Multimodal caro evitado en orquestador · Expensive multimodal avoided on orchestrator
    DeepSeek Flash v4 barato + SVP; visión solo donde hace falta (subagente).
