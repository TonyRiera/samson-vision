# Contexto y objetivo

Crear un sistema para que cualquier modelo con visión pueda actuar como lazarillo de DeepSeek v4 Flash, Nous Agent u otro modelo sin visión.

No se entrena el modelo. Se le rodea con:

- Harness.
- RAG local.
- Ejemplos buenos y malos.
- Validador.
- Memoria de errores.
- Herramientas de OCR/layout/color/DOM cuando existan.
