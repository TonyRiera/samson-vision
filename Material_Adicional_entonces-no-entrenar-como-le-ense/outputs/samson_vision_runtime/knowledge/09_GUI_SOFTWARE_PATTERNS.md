# 09 — GUI Software Patterns

Recuperar cuando `domain=gui` o `software`.

Prioridades:

1. Nombre de ventana o aplicación.
2. Barra de herramientas y menús.
3. Panel lateral.
4. Área principal.
5. Diálogos, errores, warnings.
6. Botones accionables.

Acciones:

- Identificar botón primario y secundario por posición/color si visible.
- Reportar estados: disabled, selected, active, loading.
- Para errores, extraer texto literal con OCR y coordenadas.

Errores comunes:

- Recomendar clics sobre botones deshabilitados.
- Cerrar modales sin advertir pérdida de datos.
- Inventar nombres de menús.
