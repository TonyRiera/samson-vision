# 07 — Dashboard Patterns

Recuperar cuando `domain=dashboard`.

Prioridades:

1. Título y periodo.
2. Filtros activos.
3. KPI cards.
4. Gráficos principales.
5. Tablas y alertas.
6. Leyendas y ejes.

Acciones:

- Para gráficos, describir tendencia visual sin inventar valores exactos.
- Usar OCR para números visibles.
- Distinguir color semántico: rojo alerta, verde OK, amarillo warning.

Errores comunes:

- Leer valores no visibles.
- Afirmar causalidad desde una tendencia visual.
- Confundir leyenda con datos.
