# 16 — Generic Image, Photo And Chart Patterns

Recuperar cuando `domain=generic`, `photo` o `chart`.

## Foto / escena general

Prioridades:

1. Escena global.
2. Personas, objetos principales y relaciones espaciales.
3. Texto visible.
4. Riesgos, señales, obstáculos o elementos accionables.
5. Incertidumbres por blur, oclusión, distancia o recorte.

Reglas:

- No identificar personas por nombre salvo que el usuario ya lo haya dado o el texto visible lo confirme.
- No inferir emociones internas con certeza; describir postura/expresión visible.
- No inventar ubicación geográfica.

## Gráficos

Prioridades:

1. Tipo de gráfico.
2. Título.
3. Ejes y unidades legibles.
4. Leyenda.
5. Tendencias visibles.
6. Valores exactos solo si se leen.

Reglas:

- Distinguir tendencia visual de número exacto.
- Si los ejes no son legibles, no cuantificar.
- Si un color identifica serie, mapear color a etiqueta solo si la leyenda es visible.
