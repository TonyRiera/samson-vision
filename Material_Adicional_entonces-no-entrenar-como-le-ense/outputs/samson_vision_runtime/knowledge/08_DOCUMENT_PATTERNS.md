# 08 — Document Patterns

Recuperar cuando `domain=document` o `pdf`.

Prioridades:

1. Título.
2. Encabezados.
3. Párrafos legibles.
4. Tablas.
5. Figuras y captions.
6. Pie de página y numeración.

Acciones:

- Citar solo texto legible.
- Marcar páginas o recortes incompletos.
- Separar estructura del contenido.

Errores comunes:

- Resumir contenido ilegible.
- Continuar frases cortadas.
- Ignorar tablas pequeñas.
