# 02 — Color Lexicon

Familias:

- `white`
- `black`
- `gray`
- `red`
- `orange`
- `yellow`
- `green`
- `cyan`
- `blue`
- `violet`
- `magenta`
- `brown`

Modificadores:

- `very_dark`
- `dark`
- `medium`
- `light`
- `very_light`
- `pale`
- `bright`
- `muted`
- `saturated`
- `desaturated`

Ejemplos:

- `blue_dark`
- `gray_light`
- `green_bright`
- `red_muted`
- `yellow_pale`

No uses nombres poéticos si un color básico basta.
